# **AI Chatbot using Grok API**

Web-based AI chatbot built with Python and Streamlit, integrated with the Grok AI API. Supports real-time conversation with memory and intelligent responses.

- - -
**Features**

- Interactive chat UI

- AI responses via Grok-3 / Grok-3-Mini

- Maintains conversation history

- Error handling for API requests

- - - 

**Requirements**

- Python 3.8+

- Grok API key

- Libraries: streamlit, openai, python-dotenv
